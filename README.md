# 📚 College Assistant (PDF Chatbot)

A full-stack web application that allows users to upload PDF files and ask questions based on the document content.

---

## 🚀 Features

- 📄 Upload PDF files
- 🔍 Extract text from PDFs
- 💬 Ask questions from uploaded document
- ⚡ Fast keyword-based search (no paid APIs required)
- 🧠 Lightweight and beginner-friendly

---

## 🛠️ Tech Stack

**Frontend**
- React
- Axios

**Backend**
- Node.js
- Express

**Other Tools**
- Multer (file upload)
- pdf2json (PDF parsing)

---

## 📂 Project Structure
college-assistant/
│── client/ # React frontend
│── server/ # Express backend
│── uploads/ # Temporary PDF storage
│── .env # Environment variables (NOT uploaded)
